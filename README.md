# MortalEngine

该项目为个人引擎，目前正在完善渲染功能。
渲染功能为之前从未接触过的现代化渲染APIVulkan，于是在其中写下一些Demo作为学习。具体代码在Rendering/part部分， Demo具体效果如下方图所示。
